Pass and Fail Determination

Whether or not your ARIA implementation passes or fails is determined by the rules set out in the WAI-ARIA RDF. This can get confusing since the W3C documentation often seems to be inconsistent with the RDF.

For example take menuitem which has a "required context role" of either "group", "menu" or "menubar", however the RDF only lists "menubar" or "menu".
