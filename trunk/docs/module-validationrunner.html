Module: validationrunner

This sits on top of the core validator module and "drives" it to run checks on an HTML window and format the results as HTML. It is not intended for reuse and any genuinely useful utility methods should probably be refactored to another modules.

Check the conformance of ARIA roles and attributes used in on elements in this window. This is the highest level entry point to the validator; it is a convenience method to run all other checks.
