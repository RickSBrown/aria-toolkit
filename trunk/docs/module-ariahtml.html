Module: ariahtml

This module knows about implementing ARIA in HTML5. In particular it knows about implicit semantics. The rules are specified in (and loaded from) the XML file aria-html.xml The methods are "mixed-in" to the base aria toolkit class.

Members

semantic

Used to rank the native semantic strength of an HTML element.
Properties:
Name Type Description
semantic.SACRED number An HTML element with a special native purpose that makes no sense to override with an ARIA role.
semantic.STRONG number An HTML element with strong native semantics that should not be overriden with an ARIA role.
semantic.WEAK number An HTML element with weak native semantics that can be overriden with an ARIA role.
semantic.NONE number An HTML element with no native semantics that is begging for you to add an ARIA role.
